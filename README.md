# 🚀 Rust Trading Engine (Learning Project)

This project documents my journey learning Rust and systematic trading by building a backtesting engine from scratch.

## 🎯 Goals

Learn Rust through a real-world system
Understand trading strategy design and risk
Explore AI-assisted development using Cursor
⚙️ Features (Current)
CSV-based historical data input
Moving average crossover strategy
Basic trade signal generation

## 🧠 Learning Focus

Rust ownership & memory model
Structuring a modular system
Translating trading concepts into code

## 📈 Roadmap

Add transaction fees & slippage
Implement position sizing
Introduce multiple strategies (trait-based)
Add performance metrics (Sharpe, drawdown)
Parallel backtesting
Paper trading via exchange API
